## Prerequisites

docker-compose installed on your machine.

## Installation

```zsh
COMPOSE_DOCKER_CLI_BUILD=1 DOCKER_BUILDKIT=1 docker-compose build
```

## Dev

```zsh
docker-compose up
```

## Build and start app locally

```zsh
npm run build
npm run start
```
